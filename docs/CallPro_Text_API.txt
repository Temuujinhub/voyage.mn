Notes

Урт мессеж (160+ тэмдэгт) олон segment болж илгээгдэнэ. Латин бол 160, Кирилл бол 70
unique_id параметр дамжуулбал тэр утга message_id -ийн prefix болно.
Дугаарын формат: 8 оронтой утасны дугаар хангалттай (e.g. 99112233 ). 976 prefix-тэй
эсвэл +976 prefix-тэй бас ажиллана.
Олон улсын дугаар дараах форматтай байна улсын код + дугаар /жишээ: (15612767156/US/)
